## Architecture decisions

| Decision | Rationale |
|----------|-----------|
| Burke = no LLM | Voice too specific, too ideologically precise, too plot-bearing for generative content |
| Assistant = hybrid | Critical moments (Vael warning, "You named me", stage transitions) must be authored; ambient/contextual can be generated |
| Opponent AI = BT not LLM | 11.6 t/s can't drive real-time combat; LLM generates personality at spawn, BT runs it |
| On-demand viable at min spec | 8GB VRAM + 7B Q4_K_M ≈ 40–60 t/s — changes the ceiling at ship |
